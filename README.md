# Golden Waves Hotel - Management & Booking Web App

A sophisticated, full-stack hospitality platform designed for seamless hotel operations and guest management.

## 🏨 Overview
Golden Waves Hotel is a modern booking engine and administrative suite that handles everything from luxury room reservations to event hall bookings and facility scheduling. It provides a premium experience for guests and a powerful management tool for hotel staff.

## ✨ Key Features
- **Integrated Booking Engine:** Manage rooms, events, and facility bookings in one place.
- **Role-Based Admin Dashboard:** Specialized interfaces for Owners, Managers, and Receptionists.
- **Event & Venue Management:** Handle session-based bookings for wedding and party halls.
- **Facility Scheduling:** Real-time booking for hotel recreational facilities.
- **Secure Payments & Reviews:** Integrated transaction tracking and customer feedback system.
- **High-Performance UI:** Built with React 19 and Tailwind CSS for a premium feel.

## 🛠️ Technical Implementation
- **Frontend:** React 19, Vite, Tailwind CSS, React Router, Lucide React.
- **Backend:** Node.js, Express.js.
- **Database:** Supabase (PostgreSQL) with Row Level Security (RLS).
- **Storage:** Supabase Storage for high-quality media management.
